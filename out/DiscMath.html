∀p∈P . p∣ab  ⟹  p∣a ∨ p∣b(1.1)\forall p\in\mathbb{P}\ .\ p\mid ab \implies p\mid a\ \lor\ p\mid b\tag{1.1}

Generalised to:

k∣nm ∧ gcd⁡(k,n)=1  ⟹  k∣m(1.2) k\mid nm\ \land\ \gcd(k,n)=1\implies k\mid m\tag{1.2}

gcds

Basics

Remember these rules - they are very useful for proving stuff with gcds:

  1. gcd⁡(n,m)=gcd⁡(m,n)(3.1)\gcd(n, m) = \gcd(m, n) \tag{3.1}
  2. gcd⁡(gcd⁡(l,n),m)=gcd⁡(l,gcd⁡(n,m))(3.2)\gcd(\gcd(l, n), m) = \gcd(l, \gcd(n, m))\tag{3.2}
  3. gcd⁡(lm,ln)=lgcd⁡(m,n)(3.3)\gcd(lm, ln) = l\gcd(m, n)\tag{3.3}

Also, if you have gcd(a,b)gcd(a,b), it’s often useful to insert it into any other part of a gcd expression and then use these. You can use this trick for so so so many things, it makes things so much easier!

Also remember from (one of) the definition of gcd:

gcd⁡(n,m)={nif n∣mgcd⁡(rem(n,m),m)otherwise(4) \gcd(n, m) = \begin{cases} n &\text{if }n\mid m\\ \gcd(\text{rem}(n, m), m) &\text{otherwise} \end{cases}\tag{4}

so gcd⁡(n,m)=gcd⁡(n−m,m)=gcd⁡(n+km,m)(5)\gcd(n, m) = \gcd(n - m, m) = \gcd(n + km, m)\tag{5}

Proof

RTP:  CD(n+km,m)=CD(n,m)\text{CD}(n + km, m) = \text{CD}(n, m).
RTP:  ∀d∈Z+. (d∈CD(n+km,m)  ⟺  d∈CD(n,m))\forall d\in\mathbb{Z}^+.\ (d\in\text{CD}(n + km, m)\iff d\in\text{CD}(n, m)).
 i.e.  ∀d∈Z+. (d∣n+km) ∧ d∣m  ⟺  d∣n ∧ d∣m\forall d\in\mathbb{Z}^+.\ (d\mid n+km)\ \land\ d\mid m\iff d\mid n\ \land\ d\mid m.
Let d∈Z+d\in\mathbb{Z}^+.
RTP:  (d∣n+km) ∧ d∣m  ⟺  d∣n ∧ d∣m(d\mid n+km)\ \land\ d\mid m\iff d\mid n\ \land\ d\mid m.
Assume  d∣md\mid m (i.e.  m=ldm = ld).  (i)(i)
RTP:  d∣n+km  ⟺  d∣nd\mid n+km \iff d\mid n.
RTP:  id=n+km  ⟺  jd=nid = n+km \iff jd=n  for some i,j∈Zi,j\in\mathbb{Z}.
Using (1)(1), RTP:  id=n+k(ld)  ⟺  jd=nid=n+k(ld)\iff jd=n.
RTP:  (i−kl)d=n  ⟺  jd=n(i-kl)d=n\iff jd=n  which is clearly true.
Hence,   CD(n+km,m)=CD(n,m)\text{CD}(n + km, m) = \text{CD}(n, m).
Hence,   gcd⁡(n+km,m)=gcd⁡(n,m)\gcd(n + km, m) = \gcd(n, m).

egcd (Extended gcd algorithm)

Basically go down the recursive definition given in (4)(4), but also write down at each stage an equation for each argument in terms of the original arguments:
- In the original iteration, n=Nn=N and m=Mm=M. - In later iterations, we already have one of them, and rem(n,m)=n−(quo(n,m))m\text{rem}(n, m) = n - (\text{quo}(n, m))m and use the nn and mm from the previous recursion iteration.

Thus, we get gcd⁡(n,m)=kn+lm(6)\gcd(n, m) = kn + lm\tag{6}
